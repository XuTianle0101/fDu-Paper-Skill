# Local Source Evidence

This repository previously summarized a small local set of Fudan graduate thesis examples. Treat that material only as a reminder that Fudan theses often share broad academic structures such as introduction, methods/foundation, evidence chapters, chapter summaries, conclusion, references, outputs, and acknowledgements.

Do not infer a fixed discipline, chapter sequence, method stack, or professional domain from those examples. This skill is intentionally general. When the user asks for a structure, derive it from the user's research question, evidence type, degree level, department rules, and current Fudan specification.

Use this file only when a user asks whether the skill was based on local examples. For actual writing and compliance work, prefer:

- `SKILL.md`
- `references/writing-quality-checklist.md`
- `references/fudan-2026-format-checklist.md`
